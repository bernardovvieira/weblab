jQuery UI

Questões gerais:

O que é? A biblioteca jQuery UI pode ser definida com um conjunto de widgets e componentes visuais pré-criados com jQuery para simplificar o desenvolvimento de páginas web, trazendo a possibilidade de customizar os estilos dos mesmos via temas.

Para o que serve? A biblioteca proporciona abstrações de baixo nível de interação e animação, além de avançados efeitos especiais de alto nível para construir aplicações web altamente interativas.

Qual a diferença para o jQuery padrão? O jQuery é uma biblioteca de funções JavaScript que interage com o HTML, já o o jQuery UI é um conjunto de widgets e compenentes visuais pré-criados em jQuery com a possibilidade de customizar os estilos via temas.
